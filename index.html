התחלה מהירה

התחילו להשתמש ב-RepoCraft תוך דקות

1

התקנה

התקינו את RepoCraft דרך npm או pip

npm install -g repocraft
2

יצירת מאגר

צרו מאגר חדש עם התבנית הנבחרת

repocraft create my-project --template react
3

התחלת פיתוח

התחילו לפתח את הפרויקט שלכם

cd my-project && npm start

הצטרפו לקהילה

קבלו עזרה, שתפו רעיונות ותרמו לפרויקט
